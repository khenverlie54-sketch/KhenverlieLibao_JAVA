## ABOUT THE PROGRAM

This program is a simple **Campus Library System** written in Java. It allows the user to manage books by adding new entries, updating their availability, and viewing stored records. This activity was created to practice basic Java programming concepts.

## What the Program Does

* Displays a menu for user interaction
* Adds a new book (maximum of 5 books, default status is **Available**)
* Updates a book’s status to **Borrowed** or **Available** using a serial number
* Displays all books with their serial numbers and status
* Generates a report showing total, available, and borrowed books
* Handles invalid inputs properly and allows safe program exit

## Class Used

* `Lab1JavaReview.java` – contains the main program and all methods for managing books

## Java Concepts Used

Arrays, methods, loops, conditional statements, switch-case, exception handling, `Scanner` for input, and `Random` for serial number generation.
