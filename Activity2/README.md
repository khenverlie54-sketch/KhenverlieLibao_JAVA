ABOUT THE PROGRAM

This program is a simple **Book List application** written in Java. It was created to practice object-oriented programming by using a `Book` class and managing multiple book objects in an array.

 What the Program Does

* Creates several `Book` objects with a title, author, year published, and current page
* Displays the details of each book using a method
* Updates the current page of specific books
* Updates the current page of a favorite book
* Changes the author of an existing book
* Stores all books in an array and displays titles of books published after 2010

Classes Used

* `Book` – represents a book with its details (title, author, year, current page)
* `MyBookList` – contains the main method and handles all book operations

Java Concepts Used

Classes and objects, constructors, arrays, methods, loops, conditional statements.
