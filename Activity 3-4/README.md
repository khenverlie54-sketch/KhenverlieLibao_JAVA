ABOUT THE PROGRAM

This program is a **Coffee Shop Rewards System** written in Java. It simulates how a coffee shop tracks customer reward points and allows customers to redeem free drinks. The activity was created to practice object-oriented programming concepts using a real-world scenario.

 What the Program Does

* Creates customer accounts using default and parameterized constructors
* Tracks reward points earned by each customer
* Allows customers to redeem a free drink for every 100 points
* Keeps track of the total number of registered customers using a static variable
* Displays customer information and total customer count
* Calculates how many times a customer can redeem rewards

Classes Used

* `Customer` – represents a customer with reward points and includes both instance and static methods
* `CoffeeShop` – contains the main method and demonstrates the rewards system

Java Concepts Used

Default and parameterized constructors, `this` keyword, static variables and methods, classes and objects, methods, conditional statements, and basic input/output.
